# Minesweeper
Simple Minesweeper game made with python and pygame

## How to run
1. Clone the repo
   ```bash
   git clone https://github.com/Chiraagkv/Minesweeper.git
   cd Minesweeper
   ```
2. Install dependecies
   ```bash
   pip install -r requirements.txt
   ```
3. Run `main.py`
   ```bash
   python3 main.py
   ```
